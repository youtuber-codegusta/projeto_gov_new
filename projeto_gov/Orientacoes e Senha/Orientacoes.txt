Estou apenas reforçando as orientações para a configuração do ambiente do projeto React Native, 
embora esteja ciente de que o senhor já possui familiaridade com o processo. Seguem os passos 
detalhados para garantir uma avaliação sem contratempos:

1. Configuração do Endereço IP:

Acesse o arquivo .env localizado na raiz do projeto.
Atualize a entrada do endereço IP com o valor correspondente ao servidor onde o backend está executando.


2. Atualização do Arquivo de URL:

No arquivo url.js que está no caminho: src > utils > url.js, modifique a variável que detém o endereço IP 
para que esteja alinhada ao endereço atual do servidor.

3. Ligando o Banco de Dados:

Após incluir o IP nos dois arquivos, execute separadamente o banco de dados com o código "npm run dev" até 
que apareça a msg "Banco de dados conectado com sucesso"
